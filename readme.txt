The Change User plugin allows the administrators of a blog to log in as any user of the blog.

== Description ==

Change User lets the administrators of a blog log in as any user of the blog.

When the plugin is installed and activated, a 'Log in' link is placed under each user on the user administration page.
The link will change the current user and go to the frontpage of the blog.

== Installation ==

1. Upload `change-user.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

There are no settings for the plugin.

The plugin is only tested on Wordpress 2.8.0 and newer, but might work on older versions.
